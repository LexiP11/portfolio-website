Lyft Baywheels Project Overview

This project was completed in SQLPad and Tableau Cloud. The purpose of this project was to examine the performance of Lyft's bicycle rental company, Lyft Baywheels, in the city of San Francisco. The rental company was originally Ford GoBike and Lyft bought the company from Ford in March 2020 at the height of the global pandemic. Lyft is now interested to see how they can grow their user base in San Francisco as the lockdown is coming to an end. A list of tasks that I completed in this project include:

  • Created a query to filter a dataset and union two datasets on shared columns
  • Used a SQL CASE statement to clean up a pre-existing field and stored it in a new variable
  • Performed a join in SQL to join weather data from Ford and Lyft to weather data from San Francisco
  • Created visualizations in Tableau Cloud to demonstrate the rental patterns of users based on factors such as time and weather
  • Used the visualizations to compare and contrast the user base of Ford GoBike and Lyft Baywheels
  • Provided a summary of results and recommended that Lyft should focus their marketing on commuters to expand their user base once the pandemic ends
